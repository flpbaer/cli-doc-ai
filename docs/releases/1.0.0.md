# Release 1.0.0 — 2026-04-23

### New Features

*   Initial release of `cli-doc-ai`. This command-line interface allows users to generate documentation from code comments using AI.

### Improvements

*   Improved error handling to provide more informative messages to the user.
*   Added basic configuration options for customizing the AI model used for documentation generation.
*   Enhanced the command-line help text for better usability.
